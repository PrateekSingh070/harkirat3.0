# harkirat3.0

Weekly code and exercises from **Harkirat’s cohort** track: **JavaScript**, **Node.js**, **Express**, and related backend concepts.

## Layout

- `week1/` … `week8/` — lectures and assignments grouped by week.
- Topics include HTTP servers, middleware, auth (JWT), databases, and small full-stack snippets.

## How to run

Most **Node** examples use npm. From a folder that contains a `package.json`:

```bash
npm install
node index.js
```

(or the entry file named in that week’s notes.)

## Prerequisites

- Node.js (LTS)
- npm
